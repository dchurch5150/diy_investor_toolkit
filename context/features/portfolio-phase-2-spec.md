# Portfolio UI Phase 1 Spec

## Overview

This is phase 2 of 3 for the portfolio UI layout.  Use the screenshot referenced below for how it should look.  Use a mock user for now.

## Requirements for phase 2

- Collapsible sidebar
- User avatar area at the top
- Ticker search bar below user avatar
- Links for:
  - portfolio
  - watchlists
  - analytics
  - tools
- Drawer icon to open/close sidebar.
- Always a drawer on mobile view.
