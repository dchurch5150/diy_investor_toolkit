# Portfolio UI Phase 3 Spec

## Overview

This is phase 3 of 3 for the portfolio UI layout.  Use the screenshot referenced below for how it should look.  Use a mock user for now.

## Requirements for phase 3

- Each component of the portfolio is a card
- Top row has four items:
  - Portfolio Value
  - Total Gain/Loss
  - Annual Dividend Income
  - Dividend Yield
- Next row has two items:
  - Portfolio Performance as a chart
  - Watchlist of top 5 values
- Last Row is 1 item:
  - Portfolio Holdings. table list structure of each ticker in portfolio
- Mock all data using `src/lib/mock-data.ts`
